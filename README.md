| Component | Description |
|---------|------------|
| Objective | Maximise overall customer satisfaction |
| Decision Variables | Investment in staff training, facility upgrades, dining discounts |
| Constraints | Fixed improvement budget |
| Impact Assumptions | +0.10 / +0.15 / +0.05 satisfaction per £1k invested |
| Example Allocation | £8k staff, £10k facilities, £2k discounts |
| Outcome | Satisfaction increases from ~3.5 to ~5.9 |
